How it works

This page connects to ZeroClaw's POST /v1/chat/completions endpoint (OpenAI-compatible API) via your Cloudflare tunnel.

POST {url}/v1/chat/completions
Content-Type: application/json
Authorization: Bearer zc-no-key-needed

{
  "model": "zeroclaw",
  "messages": [{"role": "user", "content": "Hello!"}]
}

The same URL works with any OpenAI-compatible app: Cursor, Continue.dev, Open WebUI, LangChain, etc.
